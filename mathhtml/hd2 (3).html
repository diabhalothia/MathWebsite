Trignometry Formulas

  • tan(x-y) = (tan x - tan y) / (1 + tan x tan y)
  • tan(x+y) = (tan x + tan y) / (1 - tan x tan y)
  • sin(2x) = 2 sin x cos x
  • cos(2x) = cos2(x) - sin2(x) = 2 cos2(x) - 1 = 1 - 2 sin2(x)
  • tan(2x) = 2 tan(x) / (1 - tan2(x))
  • sin2(x) = 1/2 - 1/2 cos(2x)
  • cos2(x) = 1/2 + 1/2 cos(2x)
